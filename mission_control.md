## 基本方針

- 稼働時間：毎日最大2時間
- **ゴール：YouTube収益化（登録者1000人 + 総再生時間4000時間）**
- 収益構造：X（拡散・認知） → Shorts（バズ・登録者獲得） → YouTube長尺（収益本体）
- 原則：Claudeによる自動化を最優先。手作業を極力ゼロにする
- 家族優先：受注型・クライアント対応が必要な仕事は不可。自分主体のみ。

**ターゲット：中高年層**（広告単価高い・視聴時間長い・固定ファンになりやすい）

**バイラル公式（10万再生から導いた）:**
```
旬のタイミング × 多数が共有する感情（怒り・驚き・共感） × 明確なスタンス = バズ
```

## システム構成（現在）

```
[サッカー速報動画パイプライン]
generate_soccer.js
  Reddit r/soccer, r/PremierLeague のPost Match Threadを検索
  Claude Haiku でマッチデータ抽出 + ナレーション生成 + 動画用JSON生成
  出力: temp/soccer_content_YYYY-MM-DD.json
      temp/soccer_video_content_YYYY-MM-DD.json

generate_soccer_video.js
  Puppeteer フレームキャプチャ（15fps）でKen Burns効果
  6スライド構成: タイトル → 解説×4 → アウトロ
  OpenAI TTS (nova) でナレーション + BGM + ビープ音
  出力: soccer_videos/soccer_YYYY-MM-DD_N.mp4 （約65秒）

[YouTube長尺（未完成）]
generate_youtube.js    ← ベース完成済み（サッカー用は未対応）

[Shorts（未完成）]
generate_shorts.js     ← 60%完成（サッカーShortsは未対応）

[X投稿（旧システム・アーカイブ済み）]
_archive_x_post/       ← sekai_no_wadai廃止に伴いアーカイブ
```
